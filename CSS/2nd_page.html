CSS pseudo-Classes

  • link - links not yet visited,
  • visited - links already visited,
  • hover - elements with mouse over them, tag:hover {}, #id:hover {}
  • focus - elements with focus on them,
  • active - elements currently being clicked,
  • x:first-child - first x element under its parent in the DOM tree,
  • x:nth-child(n|odd) - elementas x, kuris yra n-tasis vaikas, bet savo tÄ—vinio elemento
