Simple switch cases in C++ (explained)

Click to read:

This is very simple switch cases example in C++ language: 

            
                #include 
            
                    using namespace std;
                    
                    int main()
                    {
                        int day;
                    
                        cout << "Enter day no: ";
                        cin>> day;
                        switch (day)
                        {
                            case 1:cout <<"Mon";
                            break;
                            case 2:cout <<"Tue";
                            break;
                            case 3:cout <<"Wed";
                            break;
                            case 4:cout <<"Thu";
                            break;
                            case 5:cout <<"Fri";
                            break;
                            case 6:cout <<"Sat";
                            break;
                            case 7:cout <<"Sun";
                            default: cout<<"Invalid day";
                            
                            return 0;
                        }
                    }
#include this line is Standard Input / Output Streams Library.
int main() means that is main function which be executed first. Afterwards goes { ... } as everything between them means process order of the function.
int day; Describles variable used in this function and data type, int is integer that means it stores solid number.
cout << "Enter day no: "; cout<< means console out that is a rule telling debugger to promp text between " " as after every process line required we have to put ; - semicolon
cin>> day; cin>> means console in that is a rule telling debugger to take the value from user and store into variable day
switch (day) switch is a expression for variable (day) as process continues ; is NOT need it. and another {..} required as special process for switch needed.
case 1:cout <<"Mon";
break; case 1: ,
...
case 7: cout <<"Sun"; if the value of day which user inputed is one between of 1-7 the return value will be called as the output and break means that process have to be stopped.
default: cout<<"Invalid day"; this line says if cases imput number is not met (1-7), then trown out the default value which are Invalid day
